**Razbor.org** — это исследовательский проект и платформа для глубокого синтаксического и морфологического анализа русскоязычных текстов. В этом репозитории мы открыто документируем наш подход, архитектуру и трудности, с которыми столкнулись при попытке научить LLM (Большие языковые модели) понимать академические правила русского языка.

🔗 **Рабочая версия системы:** [razbor.org](https://razbor.org)

## 📌 Проблема: Почему ChatGPT не умеет делать разбор?
Современные нейросети отлично переводят тексты, пишут стихи и код. Но если попросить их сделать школьный синтаксический разбор, они с треском проваливаются. 
Причина в архитектуре: LLM предсказывает наиболее вероятное следующее слово, а не строит жесткие математические деревья зависимостей.
* ИИ галлюцинирует части речи (называет причастие прилагательным).
* ИИ теряет второстепенные члены в длинных предложениях.
* ИИ не понимает неделимых конструкций («мы с братом», «бить баклуши»).

## 🛠️ Наше решение: Гибридная архитектура
Невозможно добиться 100% точности только с помощью промпт-инжиниринга. Мы разработали конвейер (Pipeline), в котором ИИ выступает лишь как один из модулей, жестко контролируемый детерминированными алгоритмами и словарными базами.

## 🎓 Роадмап проекта
Сейчас в production работает ядро синтаксического разбора. В ближайшем будущем архитектура будет расширена для:
- Глубокого морфологического разбора (с учетом контекстной омонимии).
- Фонетического разбора (транскрипции).
- Морфемного разбора (состав слова).

## 📚 Экосистема: Прикладная лингвистика

Алгоритмы сдержек и противовесов (Smart Merge), которые мы используем для академического разбора предложений, являются фундаментом для решения повседневных задач с текстом. Глубокое понимание структуры предложения позволяет нам создавать умные утилиты для работы с русским языком.

Наш смежный проект на базе этой архитектуры:

* **[Punctuator.ru](https://punctuator.ru)** — Нейросетевой корректор орфографии. В отличие от простых словарей, он расставляет запятые, опираясь на глубокое понимание синтаксических связей, причастных оборотов и сложной иерархии предложения.
